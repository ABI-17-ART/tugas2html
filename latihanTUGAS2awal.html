Apa Itu Jurusan Pengembangan Perangkat Lunak Dan gim(PPLG)

PPLG adalah singkatan dari Pengembangan Perangkat Lunak Dan Gim dan merupakan sebuah jurusan yang ada diSekolah Menengah Kejuruan (SMK). PPLG adalah sebuah jurusan yang mempelajari dan mendalami semua semua cara-cara pengembangan perangkat lunak termasuk pembuatan, pemeliharaan, manajemen organisasi perangkat lunak dan manajemen kualitas.

Bukan hanya itu, PPLG juga berkaitan dengan software komputer mulai pembuatan website, aplikasi, game dan semua yang berkaitan dengan pemrograman dengan menguasai bahasa pemrograman tersebut. intinya RPL Tidak jauh dari tigahal yaitu Coding, Desain, dan Algoritma yang akan menjadi kunci keberhasilan Rekayasa Perangkat Lunak tersebut.

Apa Saja Yang Dipelajari Dijurusan PPLG?

Sebetulnya PPlG(Pengembangan Perangkat Lunak dan Gim) Itu Gampang! tidak jauh sama 3 hal yaitu:
  1. CODING
  2. DESAIN
  3. ALGORITMA
Didalam Coding, Desain Dan Algoritma tersebut masih ada Point-Point penting yang dipelajari di Jurusan PPLG. Berikut Daftar Peluang Kerja Jurusan PPLG:
  • Developer IT
  • Programer
  • Web Enginer
  • Game Developer
  • Database Enginer / data base administrator

    • Pemrograman Bahasa C
    • Pemrograman Bahasa C ++
    • Pemrograman BAHASA JAVA
    • Pemrograman BAHASA Python
